# Hand-written digit recognition using multi-class logistic regression

- Simple program to recognize hand-written digits from mnist data using multi-class logistic regression
- Plot the confusion matrix

## Get started

> First time


> After first time

```bash
make start
```

- Python 3+
- numpy
